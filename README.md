## Описание программы

*Book Tracker — это графическое приложение (GUI) для учёта прочитанных книг. Позволяет добавлять записи о книгах с указанием названия, автора, жанра и количества страниц, фильтровать их по жанру и объёму (количеству страниц), а также сохранять и загружать данные в формате JSON.*


Программа решает следующие задачи:
* систематизация списка прочитанных книг;
* быстрый поиск книг по жанру или размеру (например, «показать все романы длиннее 300 страниц»);
* долгосрочное хранение данных между сеансами работы;
* наглядное отображение информации в виде таблицы;
* контроль корректности вводимых данных.

## Требования

Что нужно установить на компьютер:
* Python 3.8 или выше;
* стандартная библиотека `tkinter` (входит в поставку Python);
* стандартная библиотека `json` (входит в поставку Python).
## Приложение предоставляет следующие возможности:
*Добавление книги: заполните поля «Название книги», «Автор», «Жанр», «Количество страниц» и нажмите кнопку «Добавить книгу» — запись появится в таблице.
## Фильтрация:
*по жанру: введите ключевое слово в поле «Фильтр по жанру» и нажмите «Применить фильтры»;
*по количеству страниц: укажите минимальное число страниц в поле «Фильтр страниц (>):» и нажмите «Применить фильтры».
## Сохранение данных: нажмите кнопку «Сохранить в JSON» для записи текущего списка книг в файл books.json.
*Загрузка данных: нажмите кнопку «Загрузить из JSON» для восстановления списка книг из файла books.json (если файл существует).
## Валидация ввода:
*все текстовые поля должны быть заполнены;
*поле «Количество страниц» принимает только положительные целые числа.
## Обработка ошибок:
*при повреждённом JSON‑файле выводится сообщение об ошибке;
*при некорректном вводе отображаются понятные сообщения с указанием проблемы.
## Примеры использования
## Пример 1. Добавление книги
*В поле «Название книги» введите: 1984.
*В поле «Автор» введите: Джордж Оруэлл.
*В поле «Жанр» введите: Антиутопия.
*В поле «Количество страниц» введите: 328.
*Нажмите кнопку «Добавить книгу». Книга появится в таблице ниже.
## Пример 2. Фильтрация по жанру
*Введите в поле «Фильтр по жанру»: Фантастика.
*Нажмите «Применить фильтры». В таблице останутся только книги жанра, содержащего слово «Фантастика».
## Пример 3. Фильтрация по страницам
*Введите в поле «Фильтр страниц (>): 200.
*Нажмите «Применить фильтры». В таблице останутся книги с количеством страниц 200 и более.
## Пример 4. Сохранение и загрузка
*Добавьте несколько книг.
*Нажмите «Сохранить в JSON». В папке проекта появится файл books.json.
*Закройте и перезапустите приложение.
*Нажмите «Загрузить из JSON». Список книг восстановится из файла.
## Пример 5. Обработка ошибок
*Оставьте поле «Название книги» пустым и нажмите «Добавить книгу». Появится сообщение: «Все текстовые поля должны быть заполнены!».
*Введите текст «сто» в поле «Количество страниц» и нажмите «Добавить книгу». Появится сообщение: «Количество страниц должно быть числом!».
