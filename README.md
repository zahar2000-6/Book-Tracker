# Book Tracker

**Автор:** Захар Алферов

## Описание программы

Графическое приложение для учёта прочитанных книг. Позволяет:
* добавлять книги с указанием названия, автора, жанра и количества страниц;
* фильтровать книги по жанру и количеству страниц;
* сохранять данные в JSON‑файл;
* загружать данные из JSON‑файла.

## Требования

* Python 3.8+;
* библиотека `tkinter` (входит в стандартную поставку Python).
