# Random Quote Generator

**Автор:** Захар Алферов

## Описание программы


Это GUI-приложение (с графическим интерфейсом) на Python с использованием библиотеки tkinter, которое позволяет:

генерировать случайные цитаты из базы;
фильтровать цитаты по автору и теме;
добавлять новые цитаты;
сохранять и загружать историю цитат в JSON-файл;
просматривать до 10 последних сгенерированных цитат.
Приложение предназначено для развлечения, обучения или мотивации — можно использовать дома, в офисе или на занятиях.

## Функциональность
## Основные функции
Генерация случайной цитаты — кнопка «Сгенерировать цитату» выводит случайную цитату из базы.
Фильтрация — поля «Автор» и «Тема» позволяют отфильтровать цитаты по ключевым словам.
История цитат — отображает до 10 последних сгенерированных цитат.
Добавление цитат — форма внизу окна для добавления новых цитат (поля: Текст, Автор, Тема).
Сохранение/загрузка истории — кнопки «Сохранить историю» и «Загрузить историю» работают с JSON-файлом history.json.
## Компоненты интерфейса
Верхняя часть: кнопка генерации и поля фильтрации.
Центральная часть: область вывода цитаты и список истории.
Нижняя часть: форма для добавления новых цитат.
Кнопки управления: сохранение/загрузка истории.
## Примеры использования
## Пример 1. Базовая генерация цитаты
Запустите приложение (python main.py).
Нажмите кнопку «Сгенерировать цитату».
На экране появится случайная цитата из базы.
Цитата автоматически добавится в историю (в нижней части окна).
## Пример 2. Фильтрация цитат
В поле «Автор» введите имя (например, «Лев Толстой»).
В поле «Тема» введите ключевое слово (например, «жизнь»).
Нажмите «Сгенерировать цитату» — отобразится цитата, соответствующая фильтрам.
Если совпадений нет, приложение уведомит об этом.
## Пример 3. Добавление новой цитаты
Внизу окна заполните форму:
Текст: «Будьте собой. Экспериментируйте. Выходите за рамки. Делайте ошибки. Живите полной жизнью.»
Автор: «Пабло Пикассо»
Тема: «творчество»
Нажмите кнопку «Добавить цитату».
Цитата появится в базе и может быть сгенерирована в будущем.
Поля формы очистятся автоматически.
## Пример 4. Работа с историей
Сгенерируйте 3–5 цитат.
Нажмите «Сохранить историю» — данные сохранятся в history.json.
Закройте и перезапустите приложение.
Нажмите «Загрузить историю» — предыдущие цитаты отобразятся в списке истории.
